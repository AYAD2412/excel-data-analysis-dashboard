# 📊 Excel Data Analysis Dashboard Project

## 🚀 Overview
This project demonstrates my first hands-on experience in Data Analysis using Excel.  
The goal was to transform a raw dataset into a clean, structured, and insightful dashboard.

---

## 📁 Dataset
- 234,889 rows of raw data
- Fully cleaned and structured without removing records

---

## 🧹 Data Cleaning Process
- Data cleaning using Power Query
- Handling missing values
- Fixing data types and formatting issues
- Creating new features (Year, Month, View Tiers, etc.)

---

## 📊 Analysis (Pivot Tables)
The project includes 8 Pivot Tables covering:
- Yearly performance (Views, Watch Hours, Growth %)
- Subscriber trends and cumulative analysis
- Monthly trends (63 months)
- Behavioral patterns (day/week/seasonal)
- Quarterly breakdown
- Top 211 videos ranking

---

## 📈 Dashboard Design
The dashboard is divided into 5 main sections:
- Annual Performance
- Trends Over Time
- Behavioral Patterns
- Top Video Performance
- Quarterly Breakdown

Each section combines charts + tables for better insights.
